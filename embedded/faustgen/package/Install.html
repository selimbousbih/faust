faustgen~ ============================ ## The swiss knife for [Faust](https://faust.grame.fr) development in Max/MSP. -------------- # Install Copy the `faustgen` folder to your `$HOME/Documents/Max 8/Package` folder. It works on Mac OS and Windows since Max 7. #### On Windows Assuming that MAXLOCATION is where Max is installed (e.g. usually `C:\Program Files\Cycling '74\Max 8` for Max 8), then: - copy `libsndfile-1.dll` to MAXLOCATION In case of trouble with missing dlls (msvcp140.dll, vcruntime140.dll), execute the `vc_redist.x64.exe`installer. -------------- faustgen~ development: Martin Di Rollo, St&eacute;phane Letz Help patch: Christophe Lebreton #### [Grame](http://www.grame.fr) - Centre National de Creation Musicale
