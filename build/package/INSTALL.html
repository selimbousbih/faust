# Faust ## How to install <https://faust.grame.fr> ----------------- 1) Drag the Faust folder anywhere you want. 2) Add the */faust/location*/bin folder to your `PATH`, where */faust/location* is the path to your Faust folder. Typically, you need to modify your `$HOME/.profile` or `$HOME/.bash_profile` 3) You can then open a new terminal and launch any `faust` command. ----------------- ##### Copyright (C) 2002-2021, [GRAME](http://www.grame.fr) - Centre National de Creation Musicale All rights reserved
